To install go to Builds folder and download all files of the version you want,
there is currently no installer, only a portable build.

-Functionality-
Open the Folder you want to Organize.
Choose the folder you want to save to (if the main dir is empty, you cant choose a folder).
No English version yet, coming soon.
Under "Titel" you can choose a title.
When you click open, a new Word document will be created.
This Word document will have a predefined Template with the current date and title.
It will also be saved to the folder you chose under: DDMMTitle
